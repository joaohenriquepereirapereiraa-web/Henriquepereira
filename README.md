# BN FIT — Preview static

Acesta este un preview static al aplicației Biblioteca BN FIT. Site-ul este servit ca HTML static folosind GitHub Pages.

Pentru a rula local: deschide index.html no browser (sau use um server static simples).

Site-ul folosește data/modules.json pentru conținut. Imaginile sunt placeholders (picsum.photos). Doresc să fiu notificat când vezi preview-ul și pot ajustar imaginile sau limba.
